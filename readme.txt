Plugin to restrict json rest api, which you want to restrict to use from outside

== Description ==

This Plugin Create a options which can provide ability to disable all or particular routes for json api.This plugin
help to restrict the use of public credentials of website.This will protect the data such as users details of wordpress
etc

== Installation ==

1. Download Plugin
2. Upload & Install plugin
3. Click on Disable Rest Api option.
4. Choose Disable All Json Rest Api checkbox or paste your endpoints.
5.Click on save.

In case you find difficulties in setting up your plugin, feel free to write an email to support@rajatmeshram.in.
